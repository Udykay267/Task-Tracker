# Simple To-Do List App

With this is a simple To-Do List web application built with HTML, CSS, and JavaScript. Users can:

- Add tasks
- Mark tasks as completed
- Delete tasks

 Features

- Add task: Add tasks by typing in the input field and clicking the "Add" button or pressing `Enter`.
- Mark task as completed: Mark a task as completed by clicking the ?? button.
- Delete task: Remove a task by clicking the ??? button.

 Prerequisites

No additional tools or frameworks are required to run this app. It is built entirely with front-end technologies: HTML, CSS, and JavaScript.
